# הדס תודה – אתר תדמית

אתר תדמית מודולרי עבור הדס תודה – קלינאית תקשורת.

## תכולה ראשונית
- דף בית
- אודות
- תחומי טיפול
- המלצות
- יצירת קשר
- שילוב לוגו, צבעים תואמים, טלפון, ווטסאפ

## איך מריצים מקומית
1. מתקינים את התלויות:
   ```
   npm install
   ```
2. מריצים את האתר:
   ```
   npm run dev
   ```

## איך מעלים ל-GitHub Pages
1. בונים את האתר:
   ```
   npm run build
   ```
2. מעלים ל-GitHub Pages:
   ```
   npm run deploy
   ```
